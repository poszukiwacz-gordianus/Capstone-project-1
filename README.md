# Capstone-project-1
Capstone project from The Complete 2024 Web Development Bootcamp
<hr>
Project Specifications:<br>
Create an HTML file named index.html.<br>
Use the <!DOCTYPE html> declaration at the beginning of the file to indicate that it's an HTML document.<br>
Create a head section that includes a title element with the text "My Resume".<br>
Inside the body section, write HTML code to create an online resume covering the following aspects:<br>
<ul>
  <li>Summary or objective statement</li>
  <li>Education (list degrees, schools, and dates)</li>
  <li>Work experience (list job titles, employers, dates, and responsibilities)</li>
  <li>Skills (list relevant skills or areas of expertise)</li>
  <li>Awards, certifications, or other achievements (list any relevant awards, certifications, or other accomplishments)</li>
</ul>
Use appropriate HTML tags to format the content of each section, such as h1 for section headings and ul or ol for lists.<br>
Use HTML tags to add a profile photo of yourself.<br>
Use relevant anchor tags to create a multi-page website, listing other aspects of your resume such as Hobbies and Contact details.<br>
Add a footer element with your name and any copyright information or other disclaimers. (Hint: use the MDN docs for things you don't know how to do: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/footer<br>
Save the index.html file and open it in a web browser to ensure that it displays correctly.<br>
Add your website to your GitHub to start building your portfolio.<br>
